# 소개
- SVG를 사용한 Geometry 그리기 도구
- CCTV 웹뷰어 플랫폼 개발 당시 구현했던 SVG Tool API 재현
- 재사용을 위한 `SVGGeometry` 플러그인 코드 개발 
- [데모보기](https://chodragon9.github.io/geometry/demo/)
- [데모영상](https://chodragon9.github.io/geometry/데모영상.mp4)

## 스킬셋
- 언어: ES5 / CommonJs 모듈
- 번들러: Parcel 

## 설치 및 개발 서버 실행
```bash
$ npm i
$ npm run dev
```
